# Smart Expense Tracker Dashboard

## Overview

A modern financial dashboard that helps users track income and expenses, visualize spending patterns, and gain smart insights into their financial habits.

## Features

* User authentication (login/register)
* Add, edit, and delete transactions
* Dashboard with balance, income, and expenses
* Interactive charts (category breakdown & trends)
* Smart financial insights (behavior analysis)
* Responsive fintech-style UI

## Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js
